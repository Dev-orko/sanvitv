API Documentation Summary

  • Movie: https://vidsrc-embed.ru/embed/movie?tmdb={ID}&autoplay=1
  • TV Show: https://vidsrc-embed.ru/embed/tv?tmdb={ID}&season={S}&episode={E}&autoplay=1
  • Optional Parameters: ds_lang, sub_url, autonext
  • Supports: TMDB IDs (recommended) or IMDB IDs
